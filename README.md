# S.K_mofu / K.S_mofu Visual Music Archive

Static public portfolio site for S.K_mofu / K.S_mofu.

## Contents

- `index.html`
- `styles/site.css`
- `scripts/site.js`
- `data/tracks.json`
- `assets/artwork`
- `assets/thumbs`
- `assets/teasers`

This repository is intended for GitHub Pages. Local-only sync/admin tools are intentionally not included.

## GitHub Pages

Use GitHub Pages with:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

## Local Preview

```sh
python3 -m http.server 4322
```

Then open `http://127.0.0.1:4322/`.
